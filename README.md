## Key Features


Our library management system allows you to:

1. **Add New Students and Teachers**: Easily register new students and teachers in the system, complete with their personal details.

2. **Add New Books**: Quickly add new books to the library catalog, including essential information such as title, author, and genre.

3. **Record Borrowing Transactions**: Keep track of who borrowed a particular book and when the transaction occurred. This feature ensures accountability and helps prevent loss of library resources.

4. **User-Friendly Interface**: Our app provides a beautiful and well-organized interface, making it easy for librarians to manage the library efficiently.

```sh
  cd your-prefered-folder-name
```

### Install

Install this project with:

```sh
  cd decode_morse_message

  gem install bundler

  bundle init

  Add this line to your Gemfile
    gem 'rubocop', '>= 1.0', '< 2.0'
  
  Run bundle install

  Create a new file .rubocop.yml and copy the file
  provided in the linters instructions.
```
